# Screenshots

This folder contains screenshots for documentation.

## How to Add Screenshots

1. Take screenshots of the application
2. Name them descriptively:
   - `dashboard.png` - Main dashboard view
   - `analysis.png` - Analysis results
   - `compression.png` - Compression statistics
   - `dark-mode.png` - Dark mode interface
   - `login.png` - Login screen
   - `export.png` - Export options
   - `charts.png` - Interactive charts

3. Optimize images:
   - Use PNG format for UI screenshots
   - Maximum width: 1920px
   - Compress to reduce file size
   - Use tools like TinyPNG or ImageOptim

4. Update documentation links in README.md

## Screenshot Guidelines

- **Resolution**: 1920x1080 or 1440x900
- **Format**: PNG (for UI), JPG (for photos)
- **Content**: Show real data but no sensitive information
- **Quality**: High quality, clear text
- **Annotations**: Add arrows/highlights if needed
